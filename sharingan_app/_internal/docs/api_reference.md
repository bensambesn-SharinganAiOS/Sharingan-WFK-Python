# Sharingan OS - API Reference

## Commandes Principales

### AI Uncensored
```bash
./sharingan-os.sh ai uncensored list          # Lister modèles
./sharingan-os.sh ai uncensored set <model>   # Changer modèle
./sharingan-os.sh ai uncensored query "text"  # Question AI
./sharingan-os.sh ai uncensored test          # Test connexion
```

### Agents Autonomes
```bash
./autonomous_ai_agents.sh list                # Statut agents
./autonomous_ai_agents.sh start <agent>       # Démarrer agent
./autonomous_ai_agents.sh stop-all            # Arrêter tous
```

### Tests & Qualité
```bash
./test.sh                                     # Tests complets
./deploy_ai_system.sh                        # Déploiement
```
